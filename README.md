## 🚀 Getting Started

Use this guide to download and open the website template on your computer.

1. Click the bright green button at the top that says **Download and Explore**. This will take you to the GitHub page for this project.

2. On that GitHub page, find and click the green **Code** button. This opens a dropdown menu.

3. Select **Download ZIP** from the dropdown menu. This will download all files as a single compressed folder to your computer.

4. Locate the downloaded ZIP file on your computer, usually in the **Downloads** folder.

5. Right-click the ZIP file and choose **Extract All** (Windows) or double-click it (macOS) to unzip the contents.

6. Open the new folder created after extraction. You will see several files and folders inside.

7. Find the file named `index.html`. This is the main page of the website.

8. Double-click `index.html`. Your default web browser will open the file, showing the website template.

If you want to make changes, you can open the files in a text editor and save your edits. Refresh the browser to see your updates.

## 🔧 How to Customize Your Site

This template is meant to be a starting point. Here are a few simple ways to make it your own:

- **Change Text:** Open any HTML file and replace the placeholder text with your own words.

- **Add Images:** Place your image files inside the `image` folder. Update the image links in the HTML files to point to your new images.

- **Modify Style:** Open CSS files in the `css` folder to edit colors, fonts, and layout.

- **Add Audio or Video:** Put audio and video files in the `audio` and `video` folders. Edit the HTML files to link to those media files.

- **Write Articles:** Place markdown files in the `article` folder. You may need a simple markdown viewer or convert markdown to HTML.

The structure is designed so you can add new files to match your needs without breaking the site.

## 🛠️ Common Issues and Fixes

- **The website looks broken or missing images:** Check that all images and media are in the correct folder and have the right file names.

- **Changes don’t show in browser:** Clear your browser cache or reload the page using Ctrl+F5 (Cmd+Shift+R on Mac).

- **Files won’t open:** Make sure you have unpacked the ZIP file properly.

- **HTML or CSS errors:** Simple syntax mistakes like missing quotes or tags will cause display issues. Use a text editor with syntax checking.
